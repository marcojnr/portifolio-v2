TCC Análise de Sentimento

Este projeto tem como objetivo desenvolver uma aplicação capaz de extrair automaticamente os comentários de qualquer vídeo do YouTube. Após a coleta desses dados, o sistema realiza uma análise de sentimentos com o intuito de identificar e classificar as emoções expressas nos comentários, como positivas, negativas ou neutras. Para isso, são aplicadas técnicas de machine learning, que permitem treinar modelos capazes de reconhecer padrões e comportamentos presentes no texto.
